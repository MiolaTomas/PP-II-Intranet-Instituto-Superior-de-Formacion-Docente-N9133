## Nombre del proyecto:
### Intranet - Instituto Superior de Formación Docente Nº 9133 “NUESTRA SEÑORA del CALVARIO

## Objetivos

Desarrollar una aplicación web en formato de intranet para el nivel terciario de la Escuela del Calvario que permita gestionar de manera eficiente, segura y centralizada la información académica y administrativa.

El sistema busca facilitar el acceso a la información por parte de los distintos tipos de usuarios (alumnos, profesores y personal administrativo), optimizando procesos como la inscripción a carreras, la administración de materias, la gestión de comisiones y la organización de exámenes.

---

## Descripción del proyecto

El presente proyecto tiene como objetivo el desarrollo de una aplicación web en formato de intranet para el Instituto Superior Nuestra Señora del Calvario, destinada a mejorar la gestión académica y administrativa de la institución.

El sistema permitirá el acceso mediante credenciales personales (DNI y contraseña), diferenciando el comportamiento del sistema según el tipo de usuario (alumnos, profesores y personal administrativo).

Asimismo, contará con módulos específicos para:

- Gestión de alumnos, profesores y administrativos  
- Administración de carreras, planes de estudio y materias  
- Organización de comisiones y horarios  
- Inscripción a cursado y exámenes  
- Registro de notas y condiciones académicas  

Centralizando la información y facilitando los procesos internos de la institución.

---

## Alcance

El sistema permitirá la gestión académica interna del nivel terciario dentro de un entorno de intranet institucional.

### Incluye:

- Gestión de usuarios (alumnos, profesores y administrativos)  
- Administración de carreras, materias y comisiones  
- Inscripción a cursado y exámenes  
- Gestión de mesas de examen y registro de resultados académicos  

### No incluye:

- Integración con sistemas externos  
- Acceso público desde internet (solo intranet)
